# astronomy — 천문 엔진

## 왜 사주에 천문 계산이 필요한가

사주의 기둥 경계는 달력 날짜가 아니라 **하늘의 사건**으로 정해진다. 그래서 만세력은 천문 계산을 품는다:

| 사주 요소 | 천문 사건 | 담당 파일 |
|----------|----------|----------|
| 월주(月柱) 경계 | 12절기 = 태양 황경 15°마다 | `SolarLongitude` |
| 연주(年柱) 경계 | 입춘 = 태양 황경 315° (동지설은 270°) | `SolarLongitude` |
| 시(時) 진태양시 보정 | 균시차(평균시와 진태양시의 차) | `EquationOfTime` |
| 음력 월·윤달 | 삭(朔) = 달과 태양의 황경이 같아지는 순간 | `LunarPhase` |
| 대운수 | 출생부터 다음/이전 절기까지의 일수 | `SolarLongitude` |

이 패키지는 태양·달의 위치를 VSOP87/Meeus 알고리즘으로 직접 계산한다(런타임 의존성 0). 모든 결과는 **타임존을 모르는 순수 UT 율리우스일(JD)** 이며, 한국 시간 보정은 `saju-korea` 가 맡는다(레이어 순수성).

## 읽는 순서 (알파벳 순서로 읽으면 길을 잃는다)

1. **`SolarLongitude`** — 절기·입춘·대운 경계가 여기서 나온다. *사주와 천문의 다리*이자 이 패키지의 진입점.
2. **`LunarPhase`** — 음력 삭(朔)·윤달.
3. `Ephemeris` / `SunPosition` / `MoonPosition` — 위 둘이 쓰는 위치 계산.
4. `DeltaT` / `Nutation` / `ObliquityOfEcliptic` / `AstroMath` — 정밀도를 위한 보정 항(처음엔 건너뛰어도 된다).
