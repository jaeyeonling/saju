# domain — 사주 기초 용어집

순수 불변 도메인(`java.time` 금지). 모든 enum은 한글·한자·로마자를 병기하고, 값을 암기 테이블로 박는 대신 **ordinal 산술 규칙으로 도출**한다 — 그 규칙이 곧 사주 원리다.

## 용어 한눈에

| 용어 | 코드 | 뜻 |
|------|------|-----|
| 천간(天干) | `Cheongan` | 하늘 글자 **10**: 갑을병정무기경신임계. 일주의 천간 = **일간**(日干) = 해석 기준 '나' |
| 지지(地支) | `Jiji` | 땅 글자 **12** = 12띠: 자축인묘진사오미신유술해 |
| 오행(五行) | `Ohaeng` | 목·화·토·금·수. 상생(相生, 낳음)·상극(相剋, 누름) |
| 음양(陰陽) | `Eumyang` | 양(陽)/음(陰) |
| 60갑자 | `GanZhi` | 천간 × 지지 짝이 60주기로 순환(연·월·일·시주가 모두 이 형식) |
| 4기둥(四柱) | `Pillar` / `SajuChart` | 연·월·일·시주. 각 기둥 = 천간1 + 지지1 |
| 지장간(地藏干) | `JijiHiddenStems` | 지지가 속에 품은 천간(본기/중기/여기) — 통근·격국·신강신약의 입력 |
| 자시(子時) | `ZishiPolicy` | 자정 부근(23~01시) 일주 경계를 가르는 학파(정자시/야자시) |

> 십성·십이운성·신강신약·용신·격국 등 **해석** 용어는 `saju-interpretation` 모듈에 있다.

## 도출 규칙의 예 (암기 대신 계산)

- 천간의 오행 = `ordinal / 2`, 음양 = `ordinal % 2` (`Cheongan` 정의로 고정)
- 천간합 = `(ordinal + 5) % 10` (갑기·을경·병신·정임·무계)
- 지지 육충 = `(i + 6) % 12`, 육합 = `(13 - i) % 12`
- 십성 = 오행 관계(5묶음) × 음양 동이 (`SipSeong.of`)
